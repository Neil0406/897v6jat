使用說明：

1.僅提供GET方式
2.Type: id(int)
3.Type: 其他皆為String
4.API位置請點選Link
Python範例:

python
>>> import requests
>>> import json
>>> res = requests.get('http://xx.xx.xx.xx:xx/momo_api')
>>> res
Response [200]
>>> res.json()
